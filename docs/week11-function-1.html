[課堂作業 1]：選擇輸出 1 ~ n 中全部整數、偶數、奇數、及被4整除的整數。

  • 這是期中考題目的再進化，將期中考第3題的加分題，以函數的方式設計。
  • 經由標準輸入(stdin)給定兩個參數：功能選擇及n
    • 功能選擇 = 0：輸出 1 ~ n 全部整數。
    • 功能選擇 = 1：輸出 1 ~ n 中的偶數。
    • 功能選擇 = 2：輸出 1 ~ n 中的奇數。
    • 功能選擇 = 3：輸出 1 ~ n 中可被4整除的整數。
  • 對應『功能選擇』的函數依序為：printAll(int)、printEven(int)、printOdd(int)、及printDiv4(int)。
