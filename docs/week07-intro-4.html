# GPA = 每科（學分 × 成績）的總和 / 總學分

[課堂作業 3]：「課堂作業 2」再進化，可指定每門課的學分數。 (加分題)

# GPA = 每科（學分 × 成績）的總和 / 總學分
TGP = (等第績分1) x 學分1 + (等第績分2) x 學分2 + ... + (等第績分n) x 學分n
GPA = TGP / (總學分數)

# 資料檔格式：「簡單假設」成績的順序就是科目的順序
