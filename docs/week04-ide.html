使用 git 下載複製課堂練習及作業

# 複製「整個」課堂練習及作業檔案夾到本地端電腦
git clone http://github.com/jy3736/cprog-2022s.git

# 同步更新已 "git clone" 的本地端檔案夾
git pull

在終端機裡使用 git 命令，電腦教室裡有兩種選項：

選項一：使用 Windows 終端機

  • 打開 Windows 終端機 cmd
  • 選擇工作目錄 cd
  • 下載整個課堂檔案夾 git clone
  • 同步更新課堂檔案夾 git pull

選項二：使用 VSCode 終端機

  • 打開 VSCode
  • 選擇工作目錄File → Open Folder...
  • 打開終端機 Terminal -> New Terminal
  • 下載整個課堂檔案夾 git clone
  • 同步更新課堂檔案夾 git pull

使用 VSCode + TDM-GCC 編輯及編譯程式

  • 打開 VSCode
  • 選擇工作目錄File → Open Folder...
  • 編輯程式 lab01.cpp
  • 打開終端機 Terminal -> New Terminal
  • 編譯程式 g++ -o lab01 lab01.cpp
  • 執行程式 .\lab01.exe

// File name :  lab01.cpp

#include <iostream>

using namespace std;

int main()
{
    int sum = 0;
    int n;
    cout << "Enter an integer n : ";
    cin >> n;
    for (int i = 1; i <= n; i++)
    {
        sum += i;
        cout << i << ((i < n) ? " + " : " = ");
    }
    cout << sum << endl;

    return 0;
}
